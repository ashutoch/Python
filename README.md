## 🚀 Setup

1. **Clone the repo**
```bash
   git clone https://github.com/ashutoch/your-repo-name.git
   cd your-repo-name
```

2. **Create and activate a virtual environment**
```bash
   python -m venv .venv
   .venv\Scripts\Activate.ps1   # Windows PowerShell
```

3. **Install dependencies**
```bash
   pip install -r requirements.txt
```

4. **Create a `.env` file** in the root directory with the following keys:

- GEMINI_API_KEY=your_key_here
- WEATHER_API_KEY=your_key_here
- GMAIL_ADDRESS=your_email_here
- GMAIL_APP_PASSWORD=your_app_password_here

5. **Run any script**
```bash
   python VA.py
```

## 🔒 Security Note

This repo **does not include** `.env` or `.venv` — both are excluded via `.gitignore`.
You must supply your own API keys. Never hardcode credentials in source files.
